# leanote_conf

更新leanote客户端样式，其中部分模仿了Evernote的css样式，对标题、图片显示优化等等。

**使用方法**

将该.css文件**替代** leanote/resources/app/public/themes/markdown/default/index.css 

将该.less文件**替代** leanote/resources/app/public/themes/markdown/default/index.less
